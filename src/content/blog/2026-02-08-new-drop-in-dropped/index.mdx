While working on [this very blog](https://luan.xyz), I slowly started to collect some missing features in [drop-in](https://luan.xyz/projects/drop-in); so decided to hoist some code to the upstream.

As usual, you can install the `drop-in-css` package from NPM or import it directly from:

```html
<link rel="stylesheet" href="https://luan.xyz/projects/drop-in/drop-in.css" />
```

Find the latest version on [GitHub](https://github.com/luanpotter/drop-in). Simply "drop it in" and it just works!

## New Features

<blockquote>
  You've seen these features already on this website; but now they are an official part of _drop-in_.
</blockquote>

- Better style for `<h2>` elements (seen above)
- New `.muted` class for <span class="muted">subtle text</span> (any element)
- New style for `<footer>` elements (seen below)
- New style for `<blockquote>` elements (seen above)
