Autonomous

The Autonomous is a Keyword equivalent to the following effect:
The Movement stat printed on this card is considered both transferred and not transferred. If [this] is not equipped or crewed: all transferred effects on [this] can be used as if they were not transferred. [this] can perform the Move Action. When destroyed: collect Souls according to rank.

Rulings

  • This keyword is only available for artifacts.
