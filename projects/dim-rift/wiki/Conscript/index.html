Conscript

Conscript is an Action defined as:
Summon a Creature Token to target empty active creature zone.

Rulings

  • This action is only available for creatures.
  • The Creature Token must be face-up (you can choose the orientation).
  • If the zone is no longer empty when this effect resolves: it fizzles.
