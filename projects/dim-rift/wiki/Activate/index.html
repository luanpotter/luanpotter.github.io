Activate

Activate is an Action defined as:
Select [this] or target inactive zone: activate selected zone.

Rulings

  • Note: A card might be on an inactive zone if it has Explorer or switched with a card that was put there using Explorer.
  • You cannot select an active zone.
  • You _can_ select an opponent's zone.
