Action

An Action is a slow Effect that can only be performed once per card on the field only during the Event phase. The pool of all available Actions a card can take is called its Action Pool.

Action - Effects

An Action can refer to an Effect of type Action. The following symbol printed on a card:

Specifies an additional action to be added to this card's Action Pool, meaning that this card can perform during this action during any Event Phase. It can be a simple "Action: Attack", which allows this card to use the Attack Action regardless of the Event; it can be one of the basic actions listed on the Actions page, or a unique action described on the card itself.

Action - Term

An Action can also refer to a Term used to describe a common Action present on many cards. For example, a card might say: "Action: Conscript" as a shorthand to the Conscript Action. The exact definition of each common Action can be found below.
