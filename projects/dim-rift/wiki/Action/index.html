Action

Declares an additional action that this card can perform during the Event Phase. It can be a simple "Action: Attack", which allows this card to use the Attack Action regardless of the Event chosen; it can be one of the basic actions listed on the Actions page, or a unique action described on the card.

Term

An Action can refer to a Term used to describe a common Action present on many cards. For example, a card might say: "Action: Conscript" as a shorthand to the Conscript Action. The exact definition of each common Action can be found below.
