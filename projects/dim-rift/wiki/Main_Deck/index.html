Main Deck

The Main Deck is the primary deck of cards that a player uses in a duel. It must be shuffled and its order is never altered during the game. Players must Draw cards from the top of their Main Deck. This is in opposition to the Fables Deck, which is a separate deck with always-available cards that can be summoned under certain conditions.

Decking out is not a game loss condition in Dimensional Rift; if a player attempts to draw a card from an empty Main Deck, they simply fail to draw and the game continues as normal.
