Block

Block is a potential part of a Strike when the attacking creature it is contested by another creature (as opposed to a direct attack).

There are two kinds of blocks:

  • if a creature is attacking a zone which has a creature in it, that creature must block the attack (except if it has Fading). This is called a passive block.
  • if a creature is attacking an empty zone, other creatures can decide to block the attack. This is an active block.

For passive blocks, both creatures fight simultaneously as normal. For active blocks, the attacker may have First Strike if the controller chose the Battle Event that turn. With First Strike, the attacker deals its damage first; if the active blocker is destroyed before it can retaliate, the attacker takes no damage.

Note that face-down creatures cannot actively block. If a zone with a face-down creature is targeted for an attack, that creature will first Flip due to being targeted.
