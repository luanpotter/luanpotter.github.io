Exchange

Exchange is an Action meaning:
Replace one damage counter with one mana counter on target creature, or vice-versa.

Rulings

  • This action is only available for artifacts.
  • You must choose which transaction will be performed at activation time.
  • There must be at least one counter of the type you select to exchange upon activation.
  • If there are no counters of the type selected to exchange upon resolution: the effect fizzles.
