Reap

Reap is an Action defined as:
Take a Rank 0 Soul from the Aether and put it on [this].
Reap is commonly associated with the Wynn faction.

Rulings

  • This action is only available for creatures.
  • Players can Soulrift using Souls on cards you control as if they were in the Aether.
  • If a card with Souls is destroyed, return those Souls to the Aether.
  • If a creature holding Souls is destroyed, the Souls stored and a new Soul associated with its destruction are all returned to the Aether.
