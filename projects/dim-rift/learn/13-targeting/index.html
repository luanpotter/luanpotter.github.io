Targeting

Arrows

Every non-monument card has arrows that determine what cards it connects to. A card can only target cards that are connected to it via arrows. See the example below or play with our targeting practice tool to understand how each arrow works.

Target

When a card effect uses the word target, it is a shorthand for target a zone. For example, "target creature" means "target a creature zone (with a creature in it)".

The targeted zone must be valid both during declaration and resolution, but the target itself might change. The effect will fizzle (fail to resolve), if:

  • If the targeting card no longer connects to the target zone, or has left the battlefield
  • If the card (or lack thereof) in targeted zone is no longer a valid target
