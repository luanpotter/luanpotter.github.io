Decaying Tome

Name: Decaying Tome

Type: Artifact

Faction: Unaligned

Deck: Main Deck

Rank: Rank 0

Stats: 2

Arrows:

  • topLeft: triple
  • topRight: single
  • bottomLeft: single
  • bottomRight: triple
Mandatory Once-per-turn Trigger

Recycle: Draw a card; then, discard a card.
Action
Consume any 1 mana: choose a Stat to add to _the list_ (the possible choices are: Attack, Life, Armor, Movement; _the list_ starts empty).

Consume any 1 mana: target creature gets +1 to all Stats on _the list_ until end of turn.
