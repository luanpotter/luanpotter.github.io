Hidden Tunnel Network

Name: Hidden Tunnel Network

Type: Monument

Faction: Wynn

Deck: Main Deck

Rank: Rank 0

Stats:

Arrows:

    Continuous
    Creatures and artifacts you control have +1 Movement.
    Action
    Choose any number of face-down cards of the same type you control: secretly swap them any way you desire.
    Once-per-turn Activation
    Select a face-down creature you control: flip it, and if you did: Reap.
    Reap: Separate one Soul of the lowest available rank from the Aether and put it on [this]; or, if there are no Souls in the Aether, release a Rank 0 Soul to the Aether. Players can Soulrift using Souls on cards you control as if they were in the Aether. If a card with Souls is destroyed, return those Souls to the Aether.
    Action
    Upgrade. | Erode.
    Upgrade: Sac [this] monument; summon a monument from your hand or Extra Deck in it's place, but with Cost Discount 1. Move any counters [this] had into the new monument.
    Erode: Sac [this] and another card: draw a card.
