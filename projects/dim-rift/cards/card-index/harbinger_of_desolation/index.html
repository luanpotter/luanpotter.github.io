Harbinger of Desolation

Name: Harbinger of Desolation

Type: Creature

Faction: Wynn

Deck: Extra Deck

Rank: Rank 2

Stats: 4 2 2 1 2

Arrows:

  • top: double
  • bottom: single
  • bottomRight: single
  • bottomLeft: single
Cost
Sac 2 creatures and at least 1 artifact: summon.
Mandatory Once-per-turn Trigger
When summoned: destroy target Rank 0 face-up creature.
Action
Attack.
Attack: Perform the Attack Action.
Action
Pay 1 life: Reap.
Reap: Separate one Soul of the lowest available rank from the Aether and put it on [this]; or, if there are no Souls in the Aether, release a Rank 0 Soul to the Aether. Players can Soulrift using Souls on cards you control as if they were in the Aether. If a card with Souls is destroyed, return those Souls to the Aether.
Optional Once-per-turn Trigger
When [this] is sac'ed for a summoning cost: it can count as up to 3 generic card sacrifices.
