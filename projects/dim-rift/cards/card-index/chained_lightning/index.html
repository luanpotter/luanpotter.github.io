Chained Lightning

Name: Chained Lightning

Type: Artifact - Scroll

Faction: Unaligned

Deck: Main Deck

Rank: Rank 0

Stats: 3

Arrows:

  • top: single
Mandatory Once-per-turn Trigger
When summoned: deal 2 damage to target and it consumes 2 mana. Before this effect resolves: the controller of the targeted card must target a new card from its arrows and resolve this effect. Each zone can only be targeted once during each chain.

Rulings

  • Cards must be targeted until it is not possible to continue. They key is deciding the most advantageous order to target cards.
