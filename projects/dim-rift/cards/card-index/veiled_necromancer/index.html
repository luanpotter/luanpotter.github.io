Veiled Necromancer

Name: Veiled Necromancer

Type: Creature

Faction: Wynn

Deck: Main Deck

Rank: Rank 0

Stats: 2 2 2 2

Arrows:

  • top: double
  • bottom: single
  • bottomRight: single
  • bottomLeft: single
Optional Repeatable Trigger
When flipped: Shadowwalker.
Shadowwalker: You can choose another face-down creature you control; flip this back face-down and switch their positions. You cannot activate more than one Shadowwalker effect per chain.
Optional Once-per-turn Trigger
When flipped; consume any X mana: deal X damage to target creature (note: [this] only collects mana _after_ flip effects).
Action
Pay 1 life: Reap.
Reap: Separate one Soul of the lowest available rank from the Aether and put it on [this]; or, if there are no Souls in the Aether, release a Rank 0 Soul to the Aether. Players can Soulrift using Souls on cards you control as if they were in the Aether. If a card with Souls is destroyed, return those Souls to the Aether.
Optional Repeatable Trigger
When target non-token creature you control is destroyed, consume 1 mana: summon a Creature Token in its place.
