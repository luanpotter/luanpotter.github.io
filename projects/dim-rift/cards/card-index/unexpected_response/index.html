Unexpected Response

Name: Unexpected Response

Type: Artifact - Trap

Faction: Unaligned

Deck: Main Deck

Stats: 1

Arrows:

  • top: double
  • topRight: single
  • topLeft: single
Mandatory Once-per-turn Trigger
If target card activates an Activation effect; pay 3 life: select a different Activation effect from any card on the field; resolve that effect instead, ignoring costs.
Mandatory Once-per-turn Trigger
If target card puts an effect on the stack: modify any one number from the resolution portion of the effect text by one (note: you cannot change sign but you can go to zero), and resolve the new version instead.
