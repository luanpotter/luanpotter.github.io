Mana Rock

Name: Mana Rock

Type: Artifact

Faction: Unaligned

Deck: Main Deck

Stats:

Arrows:

  • top: double
  • topRight: single
  • topLeft: single
  • left: single
  • right: single
  • bottom: single
Cost
Consume any 2 mana: summon.
Mandatory Once-per-turn Trigger
When summoned: Manalock target.
Action
Target either (1) gains or (2) consumes 1 mana.

Related Terms

Manalock: Target card cannot consume its own mana or activate effects that cost mana until the end of turn. If that card has an effect in this chain that would qualify, that effect fizzles.
