Catalyst Injector [Tier 2]

A simplified injector to catalyze symbols; it can increase the value of a simple symbol.
Per operation, you can consume X Charges: increases by X the value of a symbol with value 1.
If un-powered, X = 0, but symbols pass through. If a symbol with value != 1 passes through, it is voided.
