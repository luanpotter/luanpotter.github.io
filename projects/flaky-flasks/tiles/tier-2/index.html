Tier 2 - Tiles

Tallow Candle
A faint tallow candle.
Powers adjacent machines.

Catalyst Injector
A simplified injector to catalyze symbols; it can increase the value of a simple symbol.
When powered:: increment the value of a symbol with value 1.
If un-powered, value = 1 symbols pass through.
If a symbol with value != 1 passes through, it is voided.

Ethereal Mirror
Magically linked mirrors; allow symbols to teleport.
When powered:: teleport a symbol to the output of any Ethereal Mirror.
When un-powered, symbols can pass through the same Mirror.

Sublimation Chamber
A chamber to sublimate ingredients into fumes.
Per operation, void a symbol; create either a ≋ or a ≋ at the end of this Turn.
When powered:: you can create 2 fumes instead.

Boiler Apparatus
A traditional laboratory boiler setup; requires power input to melt a symbol into a shapeless fluid.

Shape Mold
A mold to cool and shape fluids into any specific shape.
