Tier 2 - Tiles

Catalyst Injector
A simplified injector to catalyze symbols; it can increase the value of a simple symbol.
Per operation, you can consume X units of charge: increases by X the value of a symbol with value 1.
If un-powered, X = 0, but symbols pass through. If a symbol with value != 1 passes through, it is voided.

Ethereal Mirror
Magically linked mirrors; allow symbols to teleport.
Per operation, you can consume a charge: teleport a symbol to an output mirror.

Light Sublimation Chamber
A chamber to sublimate ingredients into light fumes.
Per operation, void a symbol; create 1 ≋ at the end of this turn.
Per operation, you can consume a charge: you can create 2 ≋ instead.

Dark Sublimation Chamber
A chamber to sublimate ingredients into dark fumes.
Per operation, void a symbol; create 1 ≋ at the end of this turn.
Per operation, you can consume a charge: you can create 2 ≋ instead.

Temperamental Spindle
Each round, after Setup: discard X cards from your hand; generate X units of power.

Boiler Apparatus
A traditional laboratory boiler setup; requires power input to melt a symbol into a shapeless fluid.

Shape Mold
A mold to cool and shape fluids into any specific shape.
